Routers, Redundancy and Packets


Router

A router is a type of computer that sends data across the internet. For example the home network and the internet. They help sending data from one place to another, for example sending emails. Routers choose the safest and quickest path to send information.



Redundancy

Redundancy is inclusion of extra components incase one of the individual components fail. For example having more than one path between any two connected devices. If one path is blocked or broken by something, it will have another path to follow and the information will go where it should.



Packets

Packets are small information that travels around the internet. When the user sends a message, it breaks down into different packets, which each one of the packets take different paths. Once all of the packets arrive, they get put back together in the original order that like the first message.
